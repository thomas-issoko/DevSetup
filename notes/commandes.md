# Aide-mémoire des commandes Terminal & Git

## Navigation Terminal
- cd [dossier] : Changer de dossier.
- ls (ou dir sur Windows) : Lister les fichiers du dossier courant.
- mkdir [nom] : Créer un nouveau dossier.

## Flux de travail Git
1. git status : Vérifier l'état des fichiers.
2. git add . : Ajouter les changements.
3. git commit -m "Message" : Valider le commit.
4. git push : Propager sur GitHub.
